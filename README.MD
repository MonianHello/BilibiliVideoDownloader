首次运行文件可以帮你处理youtube-dl的问题，但是python还是要你自己上网找怎么安<br>
在Downloader的注释里面有链接，这个工具的用途就是方便你不用输入https://www.bilibili.com/video/av和-F -f_的参数了<br>
但是效率还是蛮高的<br>
